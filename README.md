# Публикация комиксов

Простой скрипт на Python, который выбирает случайный комикс с сайта `xkcd.com`, скачивает его и публикует в ваш Telegram-канал.

>Скрипт использует официальное API xkcd (без авторизации).

## Функционал

- Получает общее количество опубликованных комиксов.
- Случайным образом выбирает один из них.
- Скачивает изображение комикса.
- Отправляет его в указанный Telegram-чат с подписью (альтернативным текстом).

## Как установить

### 1. Необходимо получить API-ключ

Для корректной работы скрипта понадобятся два параметра:

#### Токен Telegram-бота

- Напишите роботу **@BotFather** в Telegram.
- Отправьте команду `/newbot` и следуйте его указаниям.
- Скопируйте полученный токен.
- Создайте канал и добавьте туда созданного бота в качестве ***администратора*** с правом публикации сообщений.

#### ID канала/группа в Телеграм (Chat ID)

- Перешлите любой пост из вашего канала в бот **@userinfobot**, чтобы узнать **Chat ID**.

### 2. Настройка окружения .env

Создайте в корневой директории проекта файл `.env` и добавьте туда ваши данные:

```env
TGBOT_TOKEN=ваш_токен_бота
TG_CHAT_ID= ID вашего_канала
```

### 3. Установка зависимостей

Рекомендуемая версия: Python 3.10+. Установите необходимые библиотеки командой:

```bash
pip install -r requirements.txt
```


## Запуск скрипта

Чтобы выбрать случайный комикс xkcd и отправить его в канал, выполните:

```bash
python main.py
```

После выполнения скрипт удалит временные файлы.
